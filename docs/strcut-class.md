# 结构伪类

## first-child
第一个元素

## last-child
最后一个元素

## nth-child()
第n个元素 写在括号里 **括号可以写公式 n为所有,也可以写odd为奇数 even为偶数**

## first-of-type
第一个:同类型前面的元素 

## nth-of-type
同类型第n个元素

## nth-last-child
倒数第n个元素

## nth-last-of-type
倒数第n个同类型元素

## only-of-type
没有同类型的元素

## :root
根元素

## empty
没有内容的元素
